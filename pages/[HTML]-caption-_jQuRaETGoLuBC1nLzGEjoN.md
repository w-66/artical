# \[HTML]\<caption>



## 属性

[**align**](https://developer.mozilla.org/zh-CN/docs/Web/HTML/Element/caption#attr-align "align") 属性已废弃

***







# 示例

-   示例1.0
    ```html
    <body>
        <h1>table</h1>
        <table align="center">
            <caption>表格标题</caption>
            <tr>
                <td>第一列</td>
                <td>第二列</td>
                <td>第三列</td>
            </tr>
        </table>
    </body>
    ```
