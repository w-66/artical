# \[Py]集合set

-   \[Py]`set()`是一个无序的不重复元素序列

\[Py]`set()`

-   示例：创建一个集合
    ```python
    # 创建集合的字面量语法
    set0 = {1, 2}
    #
    set2 = set(range(1, 10))
    #
    set3 = set((1, 2,2 , 2, 2, 3, 3, 2, 1))
    #
    print(set0, set2, set3)

    >>>
    {1, 2} {1, 2, 3, 4, 5, 6, 7, 8, 9} {1, 2, 3}

    ```
