# 属性

[\[HTML\]cols](\[HTML]cols_drtpAaHe1ADASkhUnrnAe4.md "\[HTML]cols")定义框架页含有多少列与列大小(每个值使用逗号分隔，值为像素或者百分比

[\[HTML\]rows](\[HTML]rows_apRiGgn5qXp7jJEEcLeKeg.md "\[HTML]rows")定义框架页含有多少行与行大小
